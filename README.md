Install Docker in Your machine

After installation run docker command in terminal by opening this root

```bash
$ docker-compose up -d db_pg
```

## Installation

```bash
$ npm install
# if You face any issue run with this one
$ npm install --force
```

## Do Seed

```bash
$ npm run seed
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
